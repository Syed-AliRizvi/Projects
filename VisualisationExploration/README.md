#MonashUniversity #Dashboard #Exploratino #Wrangling
This folder includes a project that includes the creation of a R-shiny dashboard to visualise and tell the story of the impact of US' War on Terror on Pakistan. 

The project was created as an assignment for the Data Exploration and Visualistaion units at Monash University. 

It includes data preprocessing on available datasets, exploration to identify existing trends, and creating the final visualisation project based on identified trends and analysis.
